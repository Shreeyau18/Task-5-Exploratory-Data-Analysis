# Task 5 – Exploratory Data Analysis (EDA)

## Objective
To perform exploratory data analysis on the Titanic dataset and extract meaningful insights using visual and statistical methods.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Titanic dataset from Kaggle.

## Key Insights
- Most passengers were aged between 20–40 years.
- Overall survival rate was low.
- Female passengers and first-class passengers had higher survival rates.
- Higher fare was associated with better survival.
- Passenger class and fare were key factors influencing survival.

## Outcome
This task improved my understanding of data exploration, visualization, and pattern identification.
